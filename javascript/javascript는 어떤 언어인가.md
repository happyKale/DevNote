## 자바스크립트의 탄생

1995년 웹 브라우저 시장 점유율 90%를 차지하고 있던 '넷스케이프 커뮤니케이션즈'는<br/>
<strong> 웹페이지의 보조적인 기능을 수행하기 위해 브라우저에서 동작하는 경량 프로그래밍 언어를 도입하기로 한다. </strong> <br/>
이것이 브렌던 아이크가 개발한 자바스크립트이다.
(이름이 '모카' -> '라이브스크립트' -> '자바스크립트' 로 바뀌었다.)
